Advanced 3dBulge

All the interest 3dBulge has been getting inspired me to do something more. So I created a more advanced visualization of the seismic activity.

At this point in time it is not in real time - but will be soon. It is now!

I just want to thank everyone for the kind emails I have been receiving - I have sadly not been able to reply to all of them but my goal is to do so eventually.

I want to make it clear that the accuracy of this is not 100% and should not be taken as such - this is only an approximation but good enough to realize the scale of everything.

Also I am going to write up a small 'blog' on how I did this - which will be available here later. So stay tuned!
